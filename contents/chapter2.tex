\subsection{\TeX{} 排版系统概述}

\TeX{} 是由 Donald E. Knuth 于 1984 年开发的排版系统，其核心优势在于精确的版面控制与高质量的数学公式排版\cite{knuth1984texbook}。\TeX{} 采用标记语言描述文档结构，通过编译生成最终的版式文件（通常为 PDF）。

\subsection{LaTeX 安装指南}

为了使用本模板，需要在计算机上安装 LaTeX 发行版。以下是不同操作系统的安装方法：

\subsubsection{Windows 系统}
\begin{enumerate}
\item 下载并安装 TeX Live
\begin{itemize}
   \item 访问 \url{https://tug.org/texlive/}
   \item 下载 `install-tl-windows.exe` 安装程序
   \item 运行安装程序，选择完整安装（约 6GB 空间）
   \item 安装完成后，重启电脑以更新环境变量
\end{itemize}
\item 验证TeX安装
   \begin{itemize}
      \item 打开命令提示符（CMD）
      \item 输入 `xelatex --version` 查看是否安装成功
   \end{itemize}
\end{enumerate}

\subsubsection{Mac 系统}
\begin{enumerate}
\item 下载并安装 MacTeX
\begin{itemize}
   \item 访问 \url{https://tug.org/mactex/}
   \item 下载最新版本的 MacTeX 安装包（约 4GB）
\end{itemize}
\item 验证安装
\begin{itemize}
   \item 打开终端（Terminal）
   \item 输入 `xelatex --version` 查看是否安装成功
\end{itemize}
\end{enumerate}
